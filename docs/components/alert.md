# Alert 警告
用于页面中展示重要的提示信息。

## 基础用法
页面中的非浮层元素，不会自动消失。
Alert 组件提供四种主题，由`type`属性指定，默认值为`info`。

<preview path="../demo/Alert/Basic.vue" title="基础用法" description="Alert组件的基础用法" />

## 主题
Alert 组件提供了两个不同的主题：`light`和`dark`。通过设置`effect`属性来改变主题，默认为`light`。
